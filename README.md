# Parking Management System 🚗

## Overview
The Parking Management System is designed to simplify and automate parking operations. It helps manage vehicle entry and exit records, parking slots, and parking details efficiently.

## Features
- Vehicle entry management
- Vehicle exit management
- Parking slot allocation
- Record maintenance
- User-friendly interface
- Efficient parking handling

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:
   git clone https://github.com/AbhishekPatel9305/Parking-Management.git

2. Open the project folder

3. Run the project
